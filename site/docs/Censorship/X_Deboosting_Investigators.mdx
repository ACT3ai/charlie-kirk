# X — Deboosting Investigators

**Deboosting** (reach suppression without visible removal) is harder to prove than suspension — but multiple Kirk investigators report it on X.

## Reported pattern

Compiled complaints describe:

| Observation | Investigators cited |
|-------------|---------------------|
| Kirk threads get **far lower impressions** than unrelated posts | [@ProjectConstitu](/Influencers/podcasts-project-constitution), [Ian Carroll](/People/ian-carroll), [George Webb](/Influencers/x/george-webb), others |
| **Replies and quote-posts** buried in search/feeds | Active OSINT accounts |
| Sudden reach drops after **intelligence / ballistics / FBI** topics | Commentary in master file |

**UVU official posts** ([@UVU](https://x.com/UVU/status/1965856814320300326)) reportedly remained visible while some **eyewitness and rooftop-access** threads saw reduced reach — per investigator observation, not platform logs.

## Fact-check layer

Investigation threads run alongside **debunk / fact-check** posts — e.g. [@AskPerplexity](https://x.com/AskPerplexity/status/1976908319504699489) pushing lone-gunman confirmation in replies. That is not proof of coordination; it is part of the **information battlefield** readers should track.

## Soft censorship vs hard ban

Deboosting leaves posts **technically visible** but limits spread — avoiding backlash from high-profile bans while slowing evidence propagation. Investigators recommend:

- Cross-post to **Rumble, BitChute, IPFS**, and this site
- Log **impression counts over time**
- Coordinate multi-account pattern checks
