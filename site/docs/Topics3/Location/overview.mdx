# Location

## Overview
This directory contains specific location information and analysis for the [Charlie Kirk](/People/charlie-kirk) incident.

## Goal
To analyze the primary event location (UVU outdoor amphitheater) and surrounding areas, and to assess the argument advanced by citizen investigators that the assassination required sophisticated intelligence coordination and advanced technology beyond [Tyler Robinson](/People/tyler-robinson)'s capabilities. This includes examining access points, security measures, and environmental factors that these commentators argue point away from the official lone-gunman narrative. These are contested arguments, not established findings.
