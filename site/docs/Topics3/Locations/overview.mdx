# Locations

## Overview
This directory contains comprehensive information about all locations relevant to the [Charlie Kirk](/People/charlie-kirk) incident.

## Goal
To map and analyze the locations that citizen investigators cite when arguing the assassination required coordination beyond a lone actor, including the UVU campus, Provo airport (where Israel/Egyptian aircraft were stationed), drone deployment zones, and intelligence staging areas. This includes examining the geographic relationships those commentators point to in arguing for an international scope and advance planning. Their conclusions are unproven contentions, not established findings.

## Key Areas
- Primary locations
- Secondary locations
- Geographic analysis
- Location relationships
- Site-specific evidence
