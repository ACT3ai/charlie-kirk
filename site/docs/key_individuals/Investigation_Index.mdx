1. Pick the **role** (security vs official vs distraction)
2. Read the cluster page for **case-specific questions**
3. Open the linked **People** profile for bio and sources
4. Follow outbound links into [Security_Team](/Security_Team/overview), [FBI](/FBI/overview), [Tyler_Robinson](/Tyler_Robinson/overview)
