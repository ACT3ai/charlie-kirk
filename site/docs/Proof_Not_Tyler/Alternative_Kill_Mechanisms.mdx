# Alternative Kill Mechanisms

**Proof Not Tyler** does not require naming a replacement shooter on day one — it requires showing
the **official .30-06 rooftop story fails** on ballistics, audio, identity, and timeline. If that
bar is met, investigators route to **alternative mechanisms** documented elsewhere. None of the
below is **proven**; each is a **hypothesis** with dedicated pages.

## 1. Inside tent — CornerShot / 9mm (close range)

[ProjectConstitu](https://x.com/ProjectConstitu/status/1973304018572071043) / Dr. Chris Martenson thread:

- **9mm +P** cavitation at tent
- **CornerShot**-style corner firing, shell casing rolling down tarp
- Shooter within ~**36 ft** of a recorder

→ [Tent — Shooter Theories](/Tent/Tent_Shooter_Theories), [Killer — Close Range](/Killer/Close_Range_Theories)

## 2. Exploding mic / shaped charge (chest or neck)

[Candace Owens](/People/candace-owens) / [UVU Tent](/UVU/Tent):

- **Rigged redundant lapel mic** meant for **chest** on white shirt; **neck hit** unintended
- **PETN** micro-charge explains **no rifle exit**, **sideways blood**, **necklace** residue search
- Rapid [tent paving](/Tent/Paving_Timeline_Sept_14) as trace removal

→ [Mic](/Mic/overview), [Mic vs Pendant](/Mic/Mic_vs_Pendant), [Medical — Lapel Mic](/Medical/lapel_mic_and_audio_theories)

## 3. Grey van / infrared laser (close range)

Van-behind-tunnel theories with **IR laser** reflections on audience phones.

→ [Killer — Close Range Theories](/Killer/Close_Range_Theories)

## 4. Professional / intelligence execution pattern

Commentators tie **precision**, **narrative speed**, **evidence alteration**, and **flight data**
([N1098L](/Planes/N1098L), [SU-BTT](/Planes/SU-BTT)) to **state/intel** capability — not proof
Robinson acted alone, but proof the case behaves like **beyond-lone-wolf** operations in other
controversial killings.

→ [Proof Intel Services](/Proof_Intel_Services/overview), [US Intelligence](/US_Intelligence/overview)

## 5. Drone or UAS kill — local patsy blame

Speculative; included as citizen theory routing only.
