# UVU Investigation Index

Utah Valley University is both **the physical kill zone** and an **institution** whose
GRAMA records show pre-event warnings buried, emergency alerts with placeholder text, and
post-shooting courtyard paving within days. This memo maps the threads without asserting
that any UVU employee committed a crime. Physical-site roster: courtyard tent, Losee Center
rooftop, 785 College Dr construction witness, Hardscape paving, Canon XA55 ~46 m audio.
