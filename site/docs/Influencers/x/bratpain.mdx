# BratPain — X Posts

[@BratPain](https://x.com/BratPain) is a commentary account cited in the site's
**faked-confession** analysis for one specific observation about how the alleged confession
messages were delivered. The observation is attributed reaction to hearing coverage — "alleged"
in every sense — and [Tyler Robinson](/People/tyler-robinson) is charged, not convicted.

## Claims from X posts

- Reacting (**July 9, 2026**) to preliminary-hearing coverage tagged @RealCandaceO, noted that
  roommate [Lance Twiggs](/People/lance-twiggs) reportedly **"said he got an automated message
  from Tyler"** and called that "sus af" — cited on
  [US Intelligence Assisted: Faked Confession](/US_Intelligence_Assisted/faked-confession).
  Automated or bot-like delivery, if authenticated, would undercut the "impulsive human
  confession" framing.
- The same flag appears on
  [Tyler Robinson Not Assassin: Faked Confession](/Tyler_Robinson_Not_Assassin/faked-confession),
  where it feeds the "who controlled the device/account" question already aimed at the Discord
  messages.

## Investigative value

A single narrow question with real forensic weight: what did the delivery metadata of the
Twiggs-channel messages look like? Server logs and device extractions either show live human
composition or they do not. The site preserves this alongside the counter-position that multiple
independent admissions (note, texts, Discord, verbal) make wholesale fabrication hard.
