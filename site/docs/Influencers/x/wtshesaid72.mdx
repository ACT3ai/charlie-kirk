## Investigative value

The BFRC is a fringe hypothesis, and the site says so — but it is preserved because it commits to a
**specific physical mechanism** with named site features, making it falsifiable in ways vaguer
theories are not. Hearing coverage of the autopsy (gunshot wound to the neck with bullet fragments)
is the standing objection any non-ballistic theory must answer.
