The "three separate people" post is one of the most-shared **witness-count claims** in the
Israel-suspect discourse: if true, those witnesses would be subpoena-grade; if overstated, the
claim collapses to a paraphrase of Owens. The site treats it as the test case for converting
viral summary into verifiable testimony.
