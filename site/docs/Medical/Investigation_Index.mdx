# Medical Investigation Index

Medical evidence should anchor the Charlie Kirk case — but **no public autopsy report** exists,
so investigators debate mechanism-of-injury through video, photographs, court filings, and
second-hand quotes. This memo maps the threads without providing medical advice or replacing
official records.
