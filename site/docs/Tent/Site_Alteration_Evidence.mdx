# Site Alteration Evidence

Investigators allege the **UVU tent homicide zone** was **physically altered within days** —
grass stripped, soil excavated, **pavers installed**, and (per some threads) **tunnel ceiling
re-mudded** and **tent fabric repaired**. This page catalogs **reported changes** and **forensic
categories** they may affect. We do **not** assert that alteration proves intentional obstruction
without work-order and forensic-clearance records.

## Reported physical changes

| Change | Source / notes |

| **Grass removed** | CK landscaping section; courtyard now brick/paver |
| **Topsoil excavated (~8–10 in.)** | Merrell paver-base practice vs Owens PETN-soil claim |
| **Pavers/bricks installed** | Merrell: **9/14/25** Sunday install |

for contractor account.

## Forensic trace categories at risk

Rapid hardscape may destroy or obscure:

1. **Bullet impacts** on decking, tent poles, vinyl backdrops, concrete pads.
2. **Blast residue** relevant to [shaped-charge / mic](/Mic/overview) hypotheses (PETN, etc.).
3. **Blood spatter patterns** for entry/exit direction mapping.
4. **Shell casings or fragments** alleged near tent edges ([Tent Shooter Theories](./Tent_Shooter_Theories)).
5. **Glass/plastic debris** from microphone housings ([Medical site changes](/Medical/construction_and_site_changes)).

Without public autopsy and scene photos, **site destruction compounds the medical vacuum**
([Autopsy not public](/Medical/autopsy_report_not_public)).

## Multi-phase work theory

Merrell describes **over-excavation** for pavers. Owens-aligned commentary adds a **prior government
team** stripping soil before Sunday. If true, the scene may have undergone **sequential** alteration
— not a single pass — increasing the need for **timestamped photography** and **soil-disposal chain
of custody**. See [10 Inches of Dirt Removed Before the Pavers](./Dirt_Removed_Before_Pavers) for the
first phase in detail, then [Daniel Merrell](./Daniel_Merrell_Hardscape_Utah) for the hardscape phase
that followed it.

## Media and official documentation gap

CK notes:

- KUTV/KATV reported courtyard changes **without naming contractor/authorizer** in some stories.
- UVU spokesperson **declined detailed questions** post-shooting.
- Strange event **#15**: **"Why paved over Tent area?"**

[FBI Crime Scene Paving](/FBI/Crime_Scene_Paving) documents Merrell's attribution of urgency to
**governor + FBI** via on-site coordinators — still **unconfirmed** by those offices in public releases.

## Comparison threads (illustrative only)

Commentary compares rapid UVU paving to other high-profile scenes altered quickly (e.g. reported
hosing at Thomas Crooks locations after the Trump attempt). Comparisons are **analogies**, not proof
of identical intent here.

## Citizen Investigator Claims on X

**Site alteration** is the umbrella citizen investigators use for paving, soil removal, tent teardown, and camera/angle loss.

- Highest-signal 2026 refresh: **[@MJTruthUltra](/Influencers/x/mjtruthultra)** Merrell package claiming rapid hardscape under Governor/FBI urgency.
- Parallel claims: state groundskeepers / workers present (see [State Workers](./State_Workers_Groundskeepers)); tent structure removal destroying spatial reference for acoustic models.
