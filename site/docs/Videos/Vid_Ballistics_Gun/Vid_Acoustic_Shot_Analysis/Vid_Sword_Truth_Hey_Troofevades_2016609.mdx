This is a generated or stock aerial standing in for the real site, which means the microphone positions plotted on it correspond to nothing on the ground.

Second, the numbers. **"0.4mm Algorithm Precision"** is not a meaningful claim about gunshot localisation. Sound travels roughly 343 millimetres in a **millisecond**; resolving a source position to under half a millimetre would require timing accurate to about a microsecond across the array, plus exact microphone positions, plus a still, uniform atmosphere, plus no reflections. In a hard-surfaced courtyard ringed by concrete and glass — the worst possible acoustic environment, dense with echoes — the honest uncertainty is metres, not fractions of a millimetre. The "93.5% Confidence" figure is offered with no stated method, no error model and no inputs.

So the clip should be read as what its own poster says it is: an illustration, made by an image generator, of a technique. Real TDOA work on this event exists and is contested; this graphic is not part of it and does not support it. It is included here because it circulates and is sometimes mistaken for a result.
