# Walid Mahmoud

Walid Mahmoud is a name that appears in unverified citizen-investigator notes in connection with a "foreign-lead" aircraft thread — specifically as a claimed passenger aboard an Egyptian-registered jet that flew into Provo, Utah in 2025. This site includes the name only to document that it circulates in online speculation, which this site does **not** endorse or treat as established fact.

## Connection to the Charlie Kirk Case

The name enters the case solely through the aircraft-tracking discussion on this site — see the [SU-BTU](/Planes/SU-BTU) and [Egyptian Armada](/Planes/Egyptian-Armada) pages. Those pages record that investigation notes name "Walid Mahmoud" as a passenger, while explicitly cautioning that this is an unverified online claim about a living person, that the same name appears elsewhere tied to a different aircraft, and that no wrongdoing has been established.

There is **no evidence** presented here connecting this individual to the shooting, and no charges of any kind are known. A named passenger on a flight is not, by itself, an indication of any involvement in a crime, and innocent business or travel explanations would be the ordinary expectation.
