# Pierce Beck

> **Disclaimer:** Inclusion on this page does not imply that this individual had knowledge of, participated in, or committed any illegal or immoral act. This page documents their publicly known connection to the events of September 10, 2025 for investigative reference only.

**Pierce Beck** is described in the investigation file as a **close-proximity security operator** with **executive protection experience** associated with **Shaffer Security Group (SSG)** — the firm founded by former FBI Special Agent [Greg Schaffer](/Security_Team/greg-shaffer) and linked historically to [Brian Harpole](/People/brian-harpole). Footage analyses identify Beck as a **bearded figure in a striped shirt** who **ran back toward the scene** after the shot. His role matters because September 10 used a **hybrid security model** that the file says was **not** an official SSG deployment that day — yet familiar protective-detail personnel still appear in citizen breakdowns of the courtyard response.

## Background

Beck is profiled in research notes as an SSG-affiliated operator with professional **executive protection** background. The investigation file states **private security on September 10** was provided by **Rockhouse Integrity Group LLC (DBA Integrity Security Solutions)** under Brian Harpole — and explicitly notes the event was **not** a Shaffer Security Group assignment that day, with SSG having **not worked with TPUSA since 2022** per Greg Schaffer's public statements.

Beck is listed alongside [Justin Davis](/People/justin-davis) in the file as security-company personnel **associated with the broader team** — with the open question of confirmed on-site assignment versus roster adjacency. **No allegation** that Beck acted outside his professional role.

## Connection to September 10, 2025

According to the investigation file and grok-compiled security threads:

- Beck **appears in footage running back** to the scene after the ~12:23 PM shooting — a response action noted in citizen investigator breakdowns.
- Visual IDs describe a **bearded man in a striped shirt** — **crowd-sourced description**, not an official UVU roster release.
- He is grouped under **"other security in close proximity"** separate from the five-person SUV team.
- The file emphasizes **no bodycams** on the hybrid TPUSA + UVU police setup — increasing reliance on bystander video and witness statements for Beck's positioning.
- Post-incident scrutiny threads mention SSG subcontractors in obstruction/cleanup discussions — **attributed commentary** about the industry footprint, **not** a charge against Beck personally.
