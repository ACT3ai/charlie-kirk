# SVG Slide Requirements

## Goal

Create an SVG file laid out like a PowerPoint slide.

## Specifications

- Aspect ratio: 16:9
- Solid colors only
- No gradients
- No imagery (no photos, no illustrations, no icons from external sources)

## Layout

Think of this as a slide deck frame. Two text elements are required:

### Top of the frame

Text: **Four Laws for Charlie Kirk Investigation**

This is the title bar at the top of the slide.

### Bottom of the frame

Text: **Law #1: FBI & Law Enforcement Release Files**

This runs across the bottom of the slide.

## Notes

The goal is to produce a clean SVG built from solid colors. Anything between
the top title and the bottom title is open to design choices, but it must stay
within the solid-color / no-gradient / no-imagery constraint.
