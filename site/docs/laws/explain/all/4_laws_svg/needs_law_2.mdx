# SVG Slide Requirements — Law #2

## Goal

Create an SVG file laid out like a PowerPoint slide. Match the visual
template of `law_1.svg` (same palette, same overall composition) so the
four law slides read as a coherent set.

## Specifications

- Aspect ratio: 16:9 (1600 x 900)
- Solid colors only
- No gradients
- No imagery (no photos, no illustrations, no icons from external sources)
- Palette must match `law_1.svg`:
  - Background: `#0F1B2D` (deep navy)
  - Title bars: `#C8102E` (red)
  - Accent stripes: `#D4AF37` (gold)
  - Content blocks: `#1B2D4A` (mid navy)
  - Block accents: `#C8102E` (red)
  - Text: `#FFFFFF` (white) and `#D4AF37` (gold) for sub-headlines

## Layout

Reuse the same frame as `law_1.svg`:

- Large gold number badge on the left with the digit `2`
- Vertical red divider between the badge and the right-side content
- Three stacked content blocks on the right
- Sub-headline above the blocks (gold)
- Caption below the blocks (white)
- Top title bar and bottom title bar, both red with gold accent stripes

## Required text

### Top of the frame

Text: **Four Laws for Charlie Kirk Investigation**

### Bottom of the frame

Text: **Law #2: Intelligence Services Release Files**

### Sub-headline above blocks

Text: **FORCED DISCLOSURE**

### Three content blocks (top to bottom)

1. **CIA**
2. **NSA**
3. **INTELLIGENCE COMMUNITY**

These represent the U.S. Intelligence Community agencies covered by Law 2 —
all of which must release final-output intelligence relevant to the Charlie
Kirk assassination, with redaction permitted only for currently active
human sources and active collection methods.

### Caption below blocks

Text: **Release All Intel. 30 Days. No Exceptions.**

## Notes

Anything else inside the frame is open to design choice, but it must stay
within the solid-color / no-gradient / no-imagery constraint, and it must
visually match the proportions and rhythm of `law_1.svg`.
