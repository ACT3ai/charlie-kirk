## Method (how correct_level_2 was chosen)

1. pages.csv `level=2` + `level2_section` column = authoritative Level 2 names (e.g. Timeline, Locations, Legal).
2. Explicit merge notes in pages.csv (e.g. property_locations → Locations).
3. Live section splits documented in pages.csv (e.g. security_law_enforcement → Security_Team + Law_Enforcement).
4. Content topic of trash overview + child pages when registry is ambiguous.
5. Empty string when content clearly spans multiple Level 2s with no single owner.
