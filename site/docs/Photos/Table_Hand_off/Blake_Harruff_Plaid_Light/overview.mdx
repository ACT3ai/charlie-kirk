## About This Cluster

This cluster collects the still images the investigation has filed under **Blake Harruff Plaid Light Blue**. Every entry links to its own page, where the image is shown at full size next to a written description of what it shows and how it connects to the rest of the investigation. The images come from public sources - social media posts, news coverage, official releases, and citizen research shared online.

Images are grouped here by concept rather than by date or source, so related evidence sits together.
