## About This Cluster

This cluster collects the still images the investigation has filed under **The Table Hand Off**. Every entry links to its own page, where the image is shown at full size next to a written description of what it shows and how it connects to the rest of the investigation. The images come from public sources - social media posts, news coverage, official releases, and citizen research shared online.

Images are grouped here by concept rather than by date or source, so related evidence sits together. Descriptions summarize what is visible and, where known, where the image has been used elsewhere on this site. Claims that appear inside the images themselves are the posters' claims, reported here with attribution rather than adopted as conclusions.

Open any image above to start, then use the previous and next links on each image page to walk the whole cluster without coming back to this list.
