## What This Image Shows

A mobile **Google Trends** capture with the query parameters visible in the address bar. The comparison runs "Tony graf" against "Robert lunnen", filtered to the **District of Columbia** over **1/1/25 - 10/19/25**, All categories, Web Search. In place of a chart, Google returns: "Hmm, your search doesn't have enough data to show here. Please make sure everything is spelled correctly, or try a more general term."

This null is the most important panel in the judge sequence. The same two names, the same window, the same tool: **Utah** returns a clean post-event news peak, **Israel** is claimed to return readings in May and July 2025, and **Washington D.C.** returns nothing at all. If the Israel panel is real, then a foreign bucket registered interest that the U.S. federal capital did not — which is precisely the asymmetry the wider search thread is built on.

It also cuts the other way, and both readings should be held together. The D.C. null proves that this pair of names sits at or below Google's reporting threshold in a large metropolitan bucket, which means the Israel panel is being drawn from a very small number of queries too. Nothing here identifies a searcher or establishes intent.
