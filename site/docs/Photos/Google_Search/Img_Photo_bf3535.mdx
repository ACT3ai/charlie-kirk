## What This Image Shows

A second capture of the same **Google Trends** comparison — "tyler james robinson" against "Lance Twiggs", filtered to the **Washington DC** market, custom range **1/1/25 - 9/10/25**, All categories, Web Search — with the tooltip again on **June 15, 2025**, reading **0** and **100**.

Duplicate captures of the same query are kept deliberately in this cluster. Google Trends recomputes its index as its trailing window moves, and this thread includes claims that specific results later became unavailable. Two independent screenshots of the same parameters returning the same values are the only durable record that the tool actually produced that output.

The reading itself is covered in full on the companion page. In short: a quoted full name for the man later charged shows no measurable D.C.-market interest in the eight months before the shooting, while his roommate's name reaches the ceiling in mid-June.
