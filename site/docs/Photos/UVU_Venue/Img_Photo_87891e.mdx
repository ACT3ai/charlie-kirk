## What This Image Shows

A screenshot of a public corporate registry page for **ROBINSONSTG, LLC**. The record shows company number **13402912-0160**, status **Active**, incorporation date **10 May 2020**, type domestic limited liability company, jurisdiction **Utah**, registered address **2682 S Franklin Dr, St George, UT 84790**, industry code **5313 Activities Related to Real Estate**, and a named manager and agent.

The reason this record is in the file is a rumour that circulated after the shooting: that a construction company connected to the accused's family carried out the work that paved over the courtyard. Researchers pulled registry records for several similarly named Utah entities to test that claim.

The claim did not hold up. Reporting logged in the investigation file found no credible link between the paving work and any company tied to the accused's father, and the contractor who has publicly described doing the work is a different, unrelated firm. This registry entry is retained as a record of a line of inquiry that was checked and came to nothing, and it makes no allegation about any person named on it.
