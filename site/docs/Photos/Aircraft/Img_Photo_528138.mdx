## What This Image Shows

A second **ADS-B Exchange** capture of **N888KG**, taken at a different point in the flight. The sidebar shows the same identity fields -- Hex AC3C75, LADD flag, CL30, 2013 Bombardier BD-100 Challenger 300, N888KG LLC -- but the telemetry has changed: groundspeed **359 kt**, barometric altitude **16,025 ft** and climbing, WGS84 altitude 16,775 ft, vertical rate **+3,328 ft/min**, track **358.2 degrees**, squawk **0634**, position **37.156, -111.401**.

Read against the descending capture on [Photo 39](/Photos/Aircraft/Img_Photo_a7f60f), which shows the aircraft southbound at 8,425 feet and falling near the same coordinates, this frame shows it climbing hard on a track of 358 degrees -- almost due north. Together they document a turn: down toward the Page area, then back up and northbound, which fits the reported pattern of a departure from [Provo](/Locations/Provo_Airport), a gap, and a return.

The squawk code 0634 is a discrete assignment from air traffic control rather than the 1200 code used for uncontrolled VFR flight, which indicates the aircraft was working with controllers on this segment. That detail sits awkwardly with the strongest versions of the covert-flight claim and is worth weighing honestly.
