## About This Cluster

These three screenshots capture the dispute over whether the published **SU-BTT** and **SU-BND** trip history is accurate. They show the same indexed date-and-flight tables as the Owens Sheet, posted as rebuttal or correction material by researchers checking each entry against other tracking sources. The folder name — "Says Wrong" — reflects that these images were filed as challenges to the sheet, not endorsements of it.

The dispute matters because the whole Egyptian-jet thread rests on those logs: if the entries fail verification, the claimed multi-year pattern weakens; if they hold, the pattern stands. Posters on each side attribute errors to the other, and neither reading has been independently confirmed. This cluster preserves the challenge material so readers can weigh both versions themselves.

Compare these against the originals in [Trip History](/Photos/Aircraft/Trip_History/overview), then read the site's written analysis at [SU-BTT](/Planes/SU-BTT) and [SU-BND](/Planes/SU-BND) for what is and is not established about these aircraft.
