## What This Image Shows

A scanned page from the 2019 Romanian complaint, describing how visitor identification data was passed to the local command for base access and stating that a prosecutor checked entry and exit registers in a numbered case.

This is administrative detail from a foreign legal matter with no connection to the [Charlie Kirk](/People/charlie-kirk) investigation, to the aircraft in this cluster, or to [September 10, 2025](/Timeline/Sept_10_2025). It sits in the aircraft folder because of a filing error.

The document set names individuals in connection with unproven allegations, which this site does not repeat. Only the document's nature and provenance are described here.
