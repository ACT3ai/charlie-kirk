## What This Image Shows

A Flightradar24 playback card for **SU-BND**, the second Egyptian-registered aircraft in this thread. The header reads "Playback of flight suBnD," FROM Provo (PVU), TO Provo (PVU), ACTUAL FLIGHT TIME 1:08, TIME 6:04 PM UTC. Aircraft is a **Gulfstream 4SP**, registration SU-BND, SERIAL NUMBER (MSN) **1332**. Telemetry reads barometric altitude 0 FT, ground speed 0 KTS, track 236 degrees, squawk 0000. Someone has drawn a red circle around the registration field.

A flight that departs and returns to the same field, running just over an hour, is the pattern researchers have flagged across this fleet -- local circuits rather than point-to-point transport. Similar Provo-to-Provo legs appear in the flight logs for **SU-BGM** preserved in the [5 Planes](/Photos/Aircraft/5_Planes/overview) sub-cluster.

Investigators tracking this aircraft report that it stayed in the Provo area after **SU-BTT** departed and left several days later via Goose Bay and Paris. Local flights of this kind have ordinary explanations -- maintenance test flights, crew currency, positioning -- none of which have been confirmed or ruled out, because no operator statement or manifest has been released.

See [SU-BND](/Planes/SU-BND) for the written analysis and [Egyptian Armada](/Planes/Egyptian-Armada) for the fleet-wide picture.
