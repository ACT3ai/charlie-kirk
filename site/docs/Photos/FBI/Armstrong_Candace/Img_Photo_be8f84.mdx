## What This Image Shows

A promotional graphic for **The Armstrong Williams Show**. The upper half is a large semi-transparent portrait of the host against an orange-to-purple gradient; the lower half shows three device mockups — a laptop, a desktop monitor, and a smartphone — each displaying a different screen from the programme. The set banner reads THE ARMSTRONG WILLIAMS SHOW, a lower third reads violence in America with the first letters cropped, and the show's web address appears beneath.

The graphic itself carries no case content. It is filed here because it identifies the broadcast platform through which some of the material in this cluster reached the public, and because the segment title visible on screen names the theme under which it was discussed.

The substantive item in this cluster is the [message screenshot](/Photos/FBI/Armstrong_Candace/Img_Photo_1a73aa), which should be read with the cautions set out on that page.
