## What This Image Shows

This is the same X post that appears elsewhere in this cluster, captured in a different layout with the portrait shown at full height beside the text. The post, last edited **11:26 a.m. on September 18, 2025**, argues that there is no verifiable public records trail for **Lance Twiggs**, points to a records entry for a similarly named **St. George** pediatrician associated with a large regional health system, and reasons from that toward behavioral-health funding as a motive structure.

It is a theory built by association, and each link in it is weak on inspection. Data-broker coverage of young adults is routinely thin, surname matches in a small state prove nothing, and health systems in every state run pediatrics and psychiatry programs. No filing, agency statement or court record supports any element of the chain, and nothing here alleges that any named individual acted improperly or unlawfully.

It is preserved because the post reached a wide audience and its component screenshots — the records page, the hospital directory listing, the news portrait — are filed separately here, so a reader can see the whole claim assembled and judge it. For the evidence about the roommate that is actually in the record, see [Twiggs Phone](/court/Twiggs_Phone), [Cellebrite](/court/Cellebrite) and [Lance](/Photos/Tyler_Robinson/Lance/overview).
