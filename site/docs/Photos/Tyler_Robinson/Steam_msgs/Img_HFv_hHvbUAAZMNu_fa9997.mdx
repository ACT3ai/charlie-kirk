## What This Image Shows

A page from an FBI investigative report describing an interview conducted by a Special Agent, with transcribed text messages between [**Tyler Robinson**](/People/tyler-robinson) and [**Lance Twiggs**](/People/lance-twiggs) reproduced in the body.

This is one of the source documents behind everything the public knows about the alleged confession thread. The chain runs: messages on the roommate's phone, a [**Cellebrite**](/court/Cellebrite) extraction confirmed in open court, an agent's interview and written report, then filings and media reproductions. Each link is a place where wording can shift, which is the concern behind the wording-comparison analyses filed at [Lance](/Photos/Tyler_Robinson/Lance/overview), and the reason researchers keep asking for the full extraction report with metadata, tool version and hash values rather than pages like this one.

Against that stands the fact that the defense holds the full data and is not publicly contesting its authenticity. This page presents the document, not a conclusion about it. Read [Twiggs Phone](/court/Twiggs_Phone), [Phone Extraction](/court/Phone_Extraction) and [Discord Messages](/Tyler_Robinson/Discord_Messages).
