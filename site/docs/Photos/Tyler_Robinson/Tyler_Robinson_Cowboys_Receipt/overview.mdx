## About This Cluster

An alibi claim is the sharpest thing a defense can have, and this cluster holds the image behind one that citizen investigators have circulated for **Tyler Robinson**. The artifact is a photographed customer receipt from **Cowboy's Smokehouse**, dated **9/10/25** — the day of the assassination at Utah Valley University — showing Check #211, Table 24, an order time of **8:55 p.m.** and a card payment at **9:47 p.m.**, with a total of $38.51. The server name and card number are blurred in the source video.

The claim, attributed to the people circulating the receipt, is that if the document were authenticated and the payment card matched to Robinson, it would be difficult to reconcile with the late-night text-message timeline built from **Lance Twiggs**' phone, which begins with a message at 11 p.m. on September 10. Nothing here has been established: the receipt has not been authenticated, the customer has not been identified, the distance and drive-time argument is the poster's assertion, and no filing by either party has adopted it. It circulates as a question worth answering rather than an answer, and it is included because the answer is checkable — a restaurant's own transaction record and card processor data would settle it quickly.
