## What This Image Shows

A dark-mode screenshot of a text-message thread, green sent bubbles and grey received bubbles, headed "Text Message - RCS" and beginning "Monday 4:11 PM."

The exchange runs: "Hello Chief Long - We received this message today from the student group. FYSA" followed by the quoted tip, "there is student roof access pretty close to where CK will be set up at Utah Valley (The Sorensen Student center has a couple stair cases that go up to the walkways on the roof". The reply asks, "You want access to the roof?" and the answer is: "I was told students have access above us, if this is true, it would be nice to either have it controlled access or allow one of my guys to be there as well. If possible." The response reads "I got you covered," and the thread closes at 8:29 PM with "Thank you, Sir."

If this thread is authentic, it is among the most consequential documents in the case, because it converts the rooftop question from hindsight into prior notice: someone raised student roof access near the setup position before the event, asked for it to be controlled or covered, and was told it was handled. [**Chief Jeff Long**](/People/jeff-long) led the UVU police contingent at the event. The screenshot carries no full date stamp, no phone numbers and no header identifying the participants, so it should be treated as an unverified reproduction until the underlying records are produced — which is exactly what disclosure requests on this point are seeking. Nothing here establishes that any individual failed in a duty.
