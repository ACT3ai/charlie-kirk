## What This Image Shows

A screenshot of a post from an account shown as **Sam Parker**, handle @BasedSamParker, posted an hour before capture, containing a records-request document headed "Event Confirmation."

The post reads: "CHARLIE KIRK BREAKING: New GRAMA request shows the September 10th stop was 'Tentatively' confirmed by UVU on August 21st--over a month after submitting the request and just 3 weeks prior to the event, which TPUSA publicly announced 6 days later on 8/27." The document shows "Turning Point Club-The American Comeback," "Event Last Modified: Aug 21 2025 10:52 AM," "Current Event State: Tentative," "Event Occurrences: Wed, Sep 10 2025 9:00 AM - 4:00 PM, Reserved: 7:00 AM - 6:00 PM," a head count line reading "Exp: 200, Reg: 0," redaction citations to Utah Code 63G-2-305, and a footer reading "Report Printed on Aug 21 2025" with reference "25-282 UVU:00002."

This is a primary document rather than a claim, which makes it unusually valuable. It fixes when the venue booking actually firmed up, how large the university expected the event to be, and which statutory exemptions were used to withhold portions.
