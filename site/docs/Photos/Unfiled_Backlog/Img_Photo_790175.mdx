## What This Image Shows

A light-mode screenshot of a post on X from an account shown as **Project Constitution**. The layout is a single long post with the standard avatar, handle and engagement row; no image is attached.

The post argues that the prosecution in the case against **Tyler Robinson** had a hand in selecting Robinson's own defense attorney, and characterizes that arrangement as unconstitutional and a mockery of justice. The sidecar text extraction for this file returned no usable lines, so the wording above is drawn from the visual description rather than quoted verbatim.

How defense counsel came to be appointed is a live question in the record, which separately tracks a pro bono defense arrangement and the sealing motions in the case. The claim here is the poster's characterization, not a finding by any court, and Robinson has not been convicted of anything.
