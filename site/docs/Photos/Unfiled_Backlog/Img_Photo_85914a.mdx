## What This Image Shows

A screenshot of a working spreadsheet with a black header row and 37 numbered data rows in nine columns.

The columns read "Index," "Date," "Flight" and an overlap section. The dates run from **2022-10-05** through **2025-07-07**. The flight column alternates between **SU-BTT** and **SU-BND**. The overlap columns are headed with the names **Charlie** and, in a partly cut heading, a second individual, and each row carries a window value of plus or minus 3 days and a present or absent flag; the visible flags read "No."

This is the actual method behind the aircraft-tailing claim, laid bare: a list of flights, a tolerance window, and a yes-or-no test against a person's known location. Publishing the sheet is what makes the claim falsifiable, and the flight-data rebuttal held elsewhere in this cluster attacks it on exactly these terms, arguing many of the listed flights did not occur. Note that the visible overlap flags in this capture read negative. See [Planes](/Planes/overview) and the [Aircraft photo cluster](/Photos/Aircraft/overview).
