## What This Image Shows

A screenshot of a Google Docs page, captured with the editor toolbar still visible at the top showing a 200 percent zoom, Normal text style, Arial font and size 11. The body is a chronological list of events and speaking engagements running from March to April, with three lines highlighted in bright yellow.

The text extraction sidecar returned no usable lines for this file, so the highlighted entries are not quoted here.

Working documents like this one are how researchers assemble a subject's movements before comparing them against flight records, venue bookings and public appearances.
