## What This Image Shows

An infographic titled **Tyler Robinson Case Discovery Example**, laid out as a side-by-side comparison and colour-coded — blue for the prosecution's edited clip, green for the defense's fuller segment of the same original source file.

It is a teaching diagram rather than evidence. The point it makes is structural: in a large discovery production, a single recording can be produced as more than one clip with different in and out points, and two parties working from the same underlying file can end up quoting different material with different apparent timings.

That is why the Bates numbering matters so much in this cluster. **Bates 003996-R2** carries an **R2** suffix, indicating a revision or a second version of the same exhibit, which is exactly the situation this diagram describes. Access to the unedited original — and to the custody logs it should match — is what the defense filings ask for and what researchers say has not been produced. The diagram illustrates a general problem; it does not establish that anything was improperly edited in this case.
