====================================================================
LINKING RULES (BACK BUTTONS AND INTERNAL LINKS)  --  READ CAREFULLY
====================================================================

* This Docusaurus site is configured with trailingSlash: false (see
  site/docusaurus.config.ts). That means a page served at /FBI/FBI_Investigation
  has NO trailing slash, so the browser treats /FBI/ as its base directory.

* CONSEQUENCE: a raw HTML relative link like <a href="../overview"> from
  /FBI/FBI_Investigation resolves in the browser to /overview (one level too
  high), which does NOT exist. This is exactly how the back buttons got broken.
  NEVER use a relative href in a raw HTML <a> tag.

* RULE 1 -- Header back-button on every Level 3 (and Level 4) page:
  Use an ABSOLUTE href that points at the Level 2 overview:

      <a href="/{Level2Section}/overview" style={{display:'inline-block',
      marginBottom:'1rem', padding:'0.35rem 0.9rem', background:'#1a73e8',
      color:'#fff', borderRadius:'4px', textDecoration:'none',
      fontSize:'0.9rem'}}>← {Level2 Label}</a>

  Example on any FBI Level 3 page: href="/FBI/overview" with label "← FBI".
  A sub-section overview (e.g. Medical/Hospital/overview.mdx) points its back
  button at its PARENT section overview: href="/Medical/overview", "← Medical".

* RULE 2 -- Header back-button on a Level 2 overview page:
  It points home: <a href="/" ...>← Home</a>. Absolute.

* RULE 3 -- Any raw HTML <a href> anywhere (back buttons, card buttons,
  "Related Areas" grids, "View Law" buttons) MUST be an absolute path that
  starts with "/". Never "../" and never a bare "overview".

* RULE 4 -- Markdown links are the exception. A markdown doc-to-doc link like
  [FBI Investigation](./FBI_Investigation) is fine and preferred, because
  Docusaurus resolves it against the source FILE path at build time, not
  against the browser URL. Overview TOC bullets should use this markdown form.

* SELF-CHECK before finishing: grep the files you touched for
  href="../  -- if any raw HTML anchor uses a relative href, rewrite it to an
  absolute /{Section}/... path. There must be zero relative raw-HTML hrefs.
