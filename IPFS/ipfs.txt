# IPFS files for the Charlie Kirk investigation
# Run these commands on any computer with IPFS installed to pull down
# all files and pin them so you rebroadcast them to others.
#
# One command per file:
#   ipfs get <CID> && ipfs pin add <CID>
#
#   ipfs get      — downloads the file to disk as a regular file
#   ipfs pin add  — fetches the content into your local IPFS node and pins it
#                   so it is never garbage-collected and you rebroadcast to peers
#   (ipfs add is not needed — pin add already stores the content locally)
